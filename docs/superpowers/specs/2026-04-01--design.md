# USMNT vs Portugal Preview — "Broadcast Premium" Redesign

## Overview

Redesign the visual layer of the USMNT vs Portugal match preview site to achieve a **premium broadcast editorial** aesthetic. The site should feel like a high-energy TV pregame show crossed with UEFA-level polish and The Athletic's typographic sophistication.

The existing React single-file architecture (index.html) is preserved. Changes focus on styling, visual hierarchy, imagery, and purposeful animation.

## Design Principles

1. **Three section tiers** — Showpieces (hero, lineups pitch, prediction), Feature sections (stats, players, H2H), Utility sections (injuries, how to watch). Each tier gets proportional visual investment.
2. **Visual variety** — Alternating backgrounds, glass-morphism cards, full-bleed sections, recessed panels. No two adjacent sections should look the same.
3. **Real imagery** — Stadium background photo, team crest SVGs/images throughout, CSS-drawn pitch diagram.
4. **Purposeful animation** — Scroll-triggered fades, staggered reveals, count-up numbers, hover lifts. Between subtle polish and broadcast energy — animations enhance, never distract.
5. **Typography contrast** — Oversized bold numbers (Bebas Neue) vs. clean body text (Inter) creates energy and scanability.
6. **Team color threading** — USA navy (#002868) and Portugal green (#006600) used consistently as accents, never overwhelming.
7. **Equal mobile/desktop priority** — Both experiences should feel great. No content hidden on mobile, layouts adapt gracefully.

## Section-by-Section Design

### 1. Top Bar

- Frosted-glass treatment (`backdrop-filter: blur`) over content
- Subtle gradient border on the bottom edge
- Broadcast chyron feel — compact, informational

### 2. Hero Section (Showpiece)

- **Full-width stadium background image** (Mercedes-Benz Stadium, Atlanta) with dark gradient overlay for text readability
- **Real team crest images** (USA Soccer, Portugal FPF) at 120-140px with drop shadows and team-color glows
- **"VS" element** with glowing pulse animation in accent yellow (#FFD200) and text-shadow bloom
- Match info (date, time, venue) in a clean row with subtle divider dots below
- Tall padding, cinematic proportions — this section breathes
- Subtle parallax effect on background image during scroll

### 3. Countdown Section

- Overlaps hero slightly with negative margin to feel connected
- **Animated count-up** — numbers start at 0 and count up when scrolled into view
- Each countdown box uses **glass-morphism** (semi-transparent background, blur, subtle border glow)
- Most urgent countdown ("2 final auditions") gets accent highlight or pulse treatment

### 4. Quick Stats Row

- Each stat in its own card with a **top accent border** in alternating colors (yellow, USA blue, Portugal green, white)
- Key number oversized (48-56px, Bebas Neue) with small label underneath
- Subtle scale-up on hover
- **Staggered fade-in-up animation** on scroll (each card delays ~100ms after previous)

### 5. Match Context

- **Different background tone** (#111116) to break visual monotony from surrounding sections
- **Two-column layout**: left column features World Cup countdown with large "73" visual anchor, right column has roster/audition context
- Recent results use inline colored pill badges (W green, L red)
- Thin vertical accent line on left edge (broadcast sidebar graphic feel)

### 6. Form Guide

- W/D/L boxes larger and more pronounced — rounded corners, subtle shadows
- **Connector line** between boxes (timeline/sparkline feel)
- Sequential pop-in animation on scroll
- Real team crest (small, 24px) inline next to team name headers

### 7. Projected Lineups (Showpiece)

- **CSS-drawn pitch diagram** — top-down soccer field with green gradient and white field lines
- Player cards positioned at their actual formation spots (3-5-2 for USA, 4-3-3 for Portugal)
- Each player shown as a small card with name and position, connected to their pitch position
- Hover/tap expands card to show club info and status
- **Tab switch animates the pitch** — players slide out, new formation slides in
- Real team crest watermarked faintly behind each team's lineup
- **Full-bleed dark gradient background** distinct from surrounding sections
- Broadcast-style tab buttons with active tab in team color and slide indicator animation

### 8. Key Players to Watch

- **Large feature cards** — player name prominent (Oswald, bold), position and club underneath
- Stats in a horizontal row, each stat in its own mini-column
- **Team-color gradient stripe** along left or top edge of each card
- Key stat pulled out as a **callout badge** (e.g., "7 goals in 2025" in yellow accent pill)
- Staggered fade-and-slide animation on scroll
- Hover effect: card lifts with increased box-shadow
- Same broadcast-style tabs as lineups section

### 9. Injuries & Availability (Utility)

- Utilitarian and scannable — no flashy animations
- **Status indicators**: red pulsing dot for OUT, amber pulsing dot for DOUBT
- Player rows have subtle left border colored by status (red/amber)
- **Recessed background** (darker) to feel like a sidebar/ticker
- Compact table layout

### 10. Head-to-Head History

- **Horizontal stacked bar chart** showing all-time W/D/L proportionally (USA blue, neutral gray, Portugal green)
- Numbers (2W-3D-2L) above their respective bar segments
- **Last 3 meetings as timeline cards** — horizontal on desktop, stacked on mobile
- Each card: large score (Bebas Neue), date and competition, one-line memorable note, left border in result color
- **2002 World Cup upset gets special treatment** — slightly larger, gold/yellow accent border, badge icon marking it as legendary
- Cards fade in left-to-right on scroll

### 11. Betting Lines

- Odds boxes become **glass-morphism cards** with percentage as a **radial progress ring** (CSS `conic-gradient`)
- Favorite (Portugal Win) gets subtle glow or "FAVORITE" badge
- Clean dashboard feel

### 12. Prediction (Showpiece)

- **Full-bleed section** with distinct background — subtle diagonal stripe pattern or dark gradient with team colors bleeding from edges
- Predicted score "1 - 2" displayed massive (80-100px, Bebas Neue), centered
- Confidence shown as an **animated arc/gauge**
- First scorer and USA scorer in accent pill badges
- Visual weight signals "this is the takeaway" — the closer of the page

### 13. How to Watch

- Two-column card layout with TV icon and radio icon treatments
- Channel names in **pill/badge style** rather than plain text
- Clean, informational

### 14. Footer

- Thin accent line on top
- Broadcast-style branding
- Subtle, doesn't compete with prediction section above

## Existing Tech Stack (Unchanged)

- Single-file React 18 app (index.html)
- Babel standalone for JSX
- React from CDN
- CSS variables for theming
- Fonts: Bebas Neue, Oswald, Inter, JetBrains Mono (Google Fonts)

## Responsive Strategy

- Equal priority for mobile and desktop
- No content hidden on mobile (reverse current approach of hiding elements)
- Pitch diagram adapts: may scale down or switch to a compact vertical layout on small screens
- Grid sections collapse from multi-column to single-column gracefully
- Touch-friendly tap targets on mobile (44px minimum)
- Animations respect `prefers-reduced-motion` media query
